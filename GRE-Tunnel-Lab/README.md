\# GRE Tunnel Lab



\## Objective



Configure a GRE tunnel between two Cisco routers using GNS3 and verify end-to-end communication.



\---



\## Technologies Used



\- Cisco IOS

\- GNS3

\- GRE Tunnel

\- IPv4

\- Static Routing
